# Pizza Bot - Desafio Técnico

Este projeto é um assistente virtual para uma pizzaria delivery, utilizando FastAPI, LangGraph, React e PostgreSQL.

## Estrutura do Projeto

- `backend/`: API FastAPI e Agente LangGraph.
- `frontend/`: Interface de Chat em React.
- `docker-compose.yml`: Orquestração dos serviços.

## Funcionalidades

-   **Chatbot Inteligente**: Conversa natural para realizar pedidos.
-   **Consulta de Preços**: O bot consulta o banco de dados para informar preços e ingredientes.
-   **Persistência**: Cardápio armazenado em banco de dados PostgreSQL.
-   **Interface Amigável**: Chat em tempo real.

## Tecnologias

-   **Backend**: Python, FastAPI, SQLModel, LangGraph, LangChain.
-   **Frontend**: React, Axios.
-   **Database**: PostgreSQL.
-   **Infra**: Docker Compose.
